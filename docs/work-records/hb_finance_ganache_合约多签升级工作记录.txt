七、风险点
- 当前合约从 constructor/immutable 改为 initializer/storage，必须重新跑完整测试和部署验证。
- 代理升级只解决后续升级入口，不能直接升级已经部署的旧 constructor 版本合约。
- implementation 地址不能给前端使用，前端和后台必须使用 proxy 地址。
- 多签成为 admin 后，所有管理操作都要通过 propose/approve/execute 执行。
