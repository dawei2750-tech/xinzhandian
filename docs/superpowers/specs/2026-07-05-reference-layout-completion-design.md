# 参考图完整布局补齐设计

## 目标

补齐参考图中的右侧摘要卡、快捷入口、Hero 指示点和利率表插画，并消除桌面 Grid 行高产生的两块空白。

## 架构

- 桌面外层为 75%/25% 两个独立纵向列，避免左右卡片互相撑高。
- 左列顶部再分 Hero 与实时数据两列，形成视觉上的三栏结构。
- 右列依次放四张摘要卡、热门币种和四张快捷入口。
- 移动端两列容器使用 `display: contents`，各区块通过 order 保持 Hero、Stats、摘要、功能、币种、利率、快捷入口、优势栏顺序。
- 新增模块的数据继续集中在 `src/constants/finance.ts`。

## 视觉补齐

- 四张摘要卡显示缩写金额与发光 SVG。
- 四张快捷入口使用不同 tone、图标、说明和箭头。
- Hero 增加轮播指示点与底部光环层。
- 两张利率表增加右下角半透明发光插画，不遮挡表格。

## 验证

- 测试新增模块、数据数量、移动端顺序和独立桌面列结构。
- PC Chrome 检查空白消失；H5 检查无横向溢出。
