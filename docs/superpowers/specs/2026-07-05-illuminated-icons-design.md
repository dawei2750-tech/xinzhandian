# HB Chain 全站点亮图标设计

## 目标

将页面中的 Unicode、字母占位图标替换为统一的项目内 SVG 图标系统，使导航、平台数据、功能卡、币种、优势栏和移动底栏接近参考图的彩色霓虹效果。

## 设计

- 新增类型安全的 `Icon` 组件，按图标名称选择 SVG path，统一使用 `currentColor`。
- 图标名称与色调继续由 `src/constants/finance.ts` 提供，组件不写业务数据。
- `GlowIcon` 负责边框、渐变底座、外发光和尺寸；小图标、标准图标、功能插画共用同一视觉系统。
- 功能卡使用较大的储蓄、礼盒、用户组和挖矿图标；其余区域使用紧凑图标。
- 所有颜色、阴影、渐变和尺寸来自 Tailwind design tokens，不使用图片、WebGL、Three.js 或外部图标库。

## 验证

- 测试所有 constants 中的图标名称均可渲染为 SVG。
- 测试页面不再出现原占位符字符。
- 在 PC 与 H5 Chrome 中确认图标可见、色调区分明确、没有溢出。
