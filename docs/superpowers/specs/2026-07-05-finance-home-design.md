# HB Chain 理财首页设计说明

## 目标

创建一个独立的 Next.js 静态前端，按桌面参考图还原深色 Web3 金融理财首页，并在 1280px、1440px 与移动端保持完整、无溢出的响应式布局。

## 范围

- 仅实现静态 UI、悬浮态和响应式布局。
- 不连接接口、钱包、后端、合约或实时行情。
- 不使用整站模板、WebGL、Three.js 或重型图表库。
- 不修改工作区中的其他项目。

## 架构

项目使用 Next.js App Router、React、TypeScript 和 Tailwind CSS。首页由专用展示组件组成，所有业务文案和表格数据集中在 `src/constants/finance.ts`，共享数据结构位于 `src/types/finance.ts`。颜色、间距、圆角、阴影与渐变统一定义为 CSS design tokens，并映射为可复用语义类，组件不写死视觉值。

## 页面结构

1. 桌面顶部导航与移动精简导航。
2. 可横向滚动的市场数据横条。
3. 左侧 Hero 与 CSS/SVG ETH 主视觉，右侧平台实时数据。
4. 四张功能卡。
5. 右侧热门币种列表和静态趋势折线。
6. 定期与灵活储蓄利率表。
7. 底部优势栏。
8. 仅移动端显示的固定底部导航。

## 视觉系统

- 背景：近黑蓝底色，叠加径向渐变、细网格和星点。
- 主色：电光蓝、紫色、青色；上涨绿色；下跌红色。
- 卡片：半透明深色背景、细边框、内阴影，hover 时增强边缘辉光。
- Hero：左文案与双按钮；右侧用 CSS/SVG 构建立体 ETH、轨道光环与 BTC、USDT、USDC 装饰。
- PC：主内容约 75%/25% 左右分栏；H5：单列，卡片纵向排列。

## 响应式规则

- 桌面最大内容宽度 1600px。
- 1280px 与 1440px 下保持双栏，窄屏压缩间距与字号。
- 小于桌面断点时切换单列；宽表格使用容器内横向滚动。
- 市场横条在移动端横向滚动且隐藏滚动条。
- 移动端预留固定底栏高度，避免内容被遮挡。
- 所有网格子项使用 `min-width: 0`，媒体与 SVG 限制在容器宽度内。

## 组件边界

- `Header`：品牌、桌面导航、语言和连接钱包静态按钮。
- `MarketTicker`：顶部市场状态横条。
- `HeroSection` / `EthVisual`：主文案和视觉装饰。
- `PlatformStats`：四项静态平台指标。
- `FeatureCards`：四项理财功能入口。
- `PopularCoins` / `TrendSparkline`：币种行情表和趋势图。
- `SavingsRateTable`：复用组件渲染两种利率表。
- `BenefitsBar`：四项平台优势。
- `MobileBottomNav`：移动端固定导航。

## 验证

- 组件测试覆盖关键文案、完整数据、利率表复用和移动导航。
- 运行测试、ESLint 与 Next.js 生产构建。
- 使用浏览器分别检查 1440px、1280px 和移动视口，确认无页面级横向溢出、布局层级与参考图一致。

## 非目标

不实现按钮业务、真实钱包连接、动态价格、服务端数据、动画引擎、用户鉴权或持久化。
